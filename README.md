# Sephara – Luxury Resort Landing Page

## Project Overview
Sephara is a luxury resort brand landing page designed to showcase premium travel destinations and encourage user reservations. The goal of this project was to create a clean, elegant, and user-friendly landing page that reflects a high-end resort experience while keeping navigation simple.

## Objective
- Design a single-page landing experience for a luxury resort brand  
- Highlight resort destinations and accommodation offerings  
- Create clear call-to-action buttons for user engagement  
- Maintain visual hierarchy, consistency, and readability  

## Tools Used
- **Figma** – UI design and layout creation  
- **Curated imagery** – for premium visual appeal  
- **Google Fonts** – clean and modern typography  
- **GitHub** – project documentation and submission  

## Design Structure

### Hero Section
The hero section introduces the brand with a strong visual banner, headline, and primary call-to-action. It is designed to immediately capture user attention and communicate luxury.

### Destinations Section
This section highlights multiple resort locations such as Lakshadweep, Maldives, and Amalfi using imagery and short descriptions to keep the layout clean and scannable.

### Accommodation Section
The accommodation section showcases the type of stay and amenities provided, using proper spacing and consistent typography for better readability.

### Testimonials Section
Testimonials are included to build user trust by displaying guest experiences in a simple and structured layout.

### Footer Section
The footer contains essential contact information and supporting links, completing the landing page structure.

## Design Considerations
- Clean and minimal layout to reflect luxury branding  
- Consistent color palette and typography  
- Proper spacing and alignment  
- Clear visual hierarchy guiding the user  

## Outcome
The final landing page delivers a polished and professional UI aligned with luxury resort branding. The design focuses on clarity, aesthetics, and user engagement, making it suitable for real-world implementation.
